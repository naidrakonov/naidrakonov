Командная строка Windows 11

wmic path SoftwareLicensingService get OA3xOriginalProductKey - запрашивает WMI для извлечения оригинального ключа продукта Windows из службы лицензирования программного обеспечения

wmic /?

sfc /scannow - проверка системных файлов на наличие повреждений и их восстановление

sfc /? - эта команда показывает справочные сведения о доступных параметрах для команды sfc

cls - очистка экрана командной строки

mkdir [имя_папки] - создать новую папку

rmdir [имя_папки] - удалить пустую папку

cd [путь] - изменить текущую директорию

copy [файлы] [папка] - копировать файлы

move [файлы] [папка] - переместить файлы

dir - показать список файлов и папок в текущей директории

ipconfig - Показывает информацию о сетевых подключениях и IP-адресах

ipconfig /?

ping [адрес] - Проверяет доступность указанного адреса в сети

ping /?

tasklist - Показывает список запущенных процессов

netstat - Показывает активные подключения и порты, которые используются на вашем компьютере

powershell - Запускает Windows PowerShell из командной строки. PowerShell предлагает более расширенные возможности по сравнению с обычной командной строкой

color /?

systeminfo - эта команда выводит полную информацию о системе, включая операционную систему, процессор, память и сетевые адаптеры. Введите команду в командной строке

wmic cpu get name, currentclockspeed, maxclockspeed - информация о оперативной памяти

wmic memorychip get capacity, speed - информация о жестких дисках

wmic diskdrive get model, size

PowerShell

Get-ComputerInfo - выводит обширную информацию о системе

Get-WmiObject Win32_Processor

Get-WmiObject Win32_Processor | Select-Object Name, NumberOfCores, NumberOfLogicalProcessors, MaxClockSpeed- выводит информацию о процессорах, включая количество, название и частоту

Get-WmiObject Win32_DiskDrive | Select-Object Model, Size, MediaType- выводит информацию о жестких дисках, таких как модель и размер

Get-WmiObject Win32_PhysicalMemory | Select-Object Capacity, Speed, Manufacturer - получите данные о модулях оперативной памяти

Get-WmiObject Win32_NetworkAdapter | Select-Object Name, NetConnectionStatus, Speed - получение информации о сетевых адаптерах

Get-WmiObject -Query "SELECT * FROM Win32_Product" | Select-Object Name, Version - просмотр списка установленных программ

Get-WmiObject Win32_Battery | Select-Object EstimatedChargeRemaining, Status, DesignCapacity - получение информации о батарее

Get-WmiObject Win32_VideoController | Select-Object Name, AdapterRAM, DriverVersion - получение информации о видеокартах, включая модель и общее количество видеопамяти

Get-LocalUser | Select-Object Name, Enabled - получение информации о текущем пользователе и его группах

Update-Help - Windows PowerShell не содержит файлов справки, однако можно просмотреть справочные разделы в Интернете или загрузить файлы справки с помощью командлета Update-Help на компьютер и затем отобразить справочные разделы с помощью командлета Get-Help в командной строке

Get-Help
